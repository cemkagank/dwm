# Simple DWM 
A minimalist dwm version of my own. Provides sane defaults with essential patches.

## Applied Patches
* `actualfullscreen`
* `attachbottom`
* `systray`
* `always center`
* `noborder`
* `hide vacant tags`
* `pertag`

## Configured for
* `playerctl`
* `st`
* `alsa`
* `xbacklight`

# Insallation
```
git clone https://github.com/cemkagank/dwm
cd dwm
sudo make install
```

#### *Remember to change your font if you do not have installed nerd-fonts*
